Sneaky Snake:
A procedurally generated game first written for the CodeDay:Portland event in February of 2015, using a custom-built JavaScript game engine: Argyle. We, GL0V Games, are prepping this game for mobile release, so stay tuned! If you have any feedback, let us know at gl0vgames {dot} studio {at} gmail {dot} com!

Objective:
Sneak around the enemies to make your way to the teleporter, without being seen, to get to the next level. If you don't see a teleporter, make sure to check all the corners! Walking over the enemies increases your score and moving speed, but also makes you longer. The resulting tail can be seen by enemies, so plan carefully, and see how long your tail can get!

Controls:
w, s, a, d to move around, if the controls are confusing try clicking the Toggle Controls button to switch them.
m to mute
r to restart
t to toggle controls
